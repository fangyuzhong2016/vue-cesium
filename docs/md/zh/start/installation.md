# 安装

## NPM

```bash
npm install vue-cesium --save
```

## YARN

```bash
yarn add vue-cesium
```

## CDN

```js
<!-- include Vue -->
<script src="https://unpkg.com/vue@latest/dist/vue.min.js"></script>
<!-- include VueCesium -->
<script src="https://unpkg.com/vue-cesium@latest/lib/index.umd.js"></script>
```
